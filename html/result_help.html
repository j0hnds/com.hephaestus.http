HTTP Response Data

The fields in this area allow you to view the results of invoking the HTTP Request. The following fields/tabs are available:

Status
The HTTP Status code returned from the server. The following are common Status Codes:
  • 100 - Continue
  • 101 - Switching Protocols
  • 200 - OK
  • 201 - Created
  • 202 - Accepted
  • 203 - Non-Authoritative Information
  • 204 - No Content
  • 205 - Reset Content
  • 206 - Partial Content
  • 300 - Multiple Choices
  • 301 - Moved Permanently
  • 302 - Found
  • 303 - See Other
  • 304 - Not Modified
  • 305 - Use Proxy
  • 306 - Reserved (not used)
  • 307 - Temporary Redirect
  • 400 - Bad Request
  • 401 - Unauthorized
  • 402 - Payment requred (reserved)
  • 403 - Forbidden
  • 404 - Not Found
  • 405 - Method Not Allowed
  • 406 - Not Acceptable
  • 407 - Proxy Authentication Required
  • 408 - Request Timeout
  • 409 - Conflict
  • 410 - Gone
  • 411 - Length Required
  • 412 - Precondition Failed
  • 413 - Request Entity Too Large
  • 414 - Request-URI Too Long
  • 415 - Unsupported Media Type
  • 416 - Requested Range Not Satisfiable
  • 417 - Expectation Failed
  • 500 - Internal Server Error
  • 501 - Not Implemented
  • 502 - Bad Gateway
  • 503 - Service Unavailable
  • 504 - Gateway Timeout
  • 505 - HTTP Version Not Supported
Data
This tab displays the data received in the response. The type of data displayed depends on what the server returns for a particular request. If the response has a Content-Type of text/*, the text of the response will be displayed. If the response Content-Type is of any other type, a hex-dump of the response data is displayed.
HTTP Headers
This tab displays the HTTP Headers returned in the response. The headers received are strictly determined by the response from the server.
