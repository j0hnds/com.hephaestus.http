HTTP Headers

You can specify the HTTP Headers you would like to send with the request. Simply select the tab folder named HTTP Headers and add the headers/values you would like to be added to the request.

  • To add a header, right-click in the HTTP Headers table and select the Insert menu item. A new row will be added to the table.
  • To modify the header values, use the mouse to select the header name or value you wish to change and start typing. When done editing the name or value, hit the Enter key.
  • To delete a header row, select the row to be deleted, then right-click in the table (on that row) and select Delete.
  • To delete all the header rows, right-click in the table, then select the Delete All menu item.
