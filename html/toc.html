Welcome to the HTTPClient Plugin

This plugin provides a visual interface for invoking a variety of different types of HTTP requests. It supports key HTTP verbs:

  • GET
  • POST
  • PUT
  • DELETE

This makes it convenient to use as a test client for REST web services.

The plugin makes it easy to set up the data for POST/PUT calls. The data can be specified in the following ways:

  • Form Post Data - This allows you to specify the data in terms of name-value pairs and submitted as if you had posted a form.
  • Bulk Post Data - This allows you to specify the textual content of the data to post along with a content-type specifying the type of data.
  • File Upload - This allows you to specify a file to upload with the POST.

For further flexibility, you can specify the HTTP headers you wish to send (above and beyond the standard HTTP headers).

The plugin also provides import/export functions to allow you to define the calls you want to make, save them and restore them at a later time, saving you precious time in setting up the call again.
