Import / Export

To make it more convenient to invoke HTTP requests over a period of time, the HTTPClient plugin allows you to save the settings for a particular HTTP request and load it at a later time.

The settings saved in the export are:

  1. Full URL
    • Verb
    • Protocol
    • Host:port
    • URI
  2. HTTP Headers
  3. POST/PUT RequestData
    • Form request data
    • Bulk Data w/content type
    • File Upload Path

To use this functionality,

  1. Set up the HTTP request as you would like it to be saved.
  2. Invoke the Export function (view tool bar or view drop-down menu "Export". A File Dialog will be displayed. Specify the location and name of the file to save. A ".xml" extension will automatically be added to the file name if necessary.

Now that the file has been saved, you can restore this request at any time by:

  1. Invoke the Import function (view tool bar or view drop-down menu "Import". A File Dialog will be displayed. Specify the location and name of the file to import. Hit OK and the plugin view will be reset with the values saved in the file.
